# Currency Converter



## Автор: Стекольников Владислав



## Описание

Графический конвертер валют с использованием внешнего API и сохранением истории.


## Как получить API-ключ

1. Зарегистрируйтесь на сайте exchangerate-api.com.

2. Получите бесплатный API-ключ в личном кабинете.

3. Вставьте ключ в переменную `API\_KEY` в файле `main.py`.



## Запуск приложения

```bash

python currency_converter.py

```



## Использование

1. Выберите валюту «Из» и «В».

2. Введите сумму.

3. Нажмите «Конвертировать».

4. Результат появится ниже, операция добавится в историю.



## История

История сохраняется в файле `history.json`. Можно просматривать в таблице внутри приложения.
